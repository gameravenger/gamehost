## Image Guidelines:

### Format Requirements
- Use WebP format when possible for better compression
- Provide fallback JPG/PNG for older browsers
- Optimize all images for web (compress without quality loss)

### Size Guidelines
- Game banners: 400x600px (portrait A4 style)
- User avatars: 200x200px (square)
- Promotional banners: 800x200px (landscape)
- Background images: 1920x1080px (full HD)

### Mobile Optimization
- Provide @2x versions for high-DPI displays
- Use responsive images with srcset attribute
- Lazy load images below the fold

### Accessibility
- Include alt text for all images
- Use descriptive filenames
- Ensure good contrast for text overlays

## Implementation Notes:
